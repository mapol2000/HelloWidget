# 위젯 widget
안드로이드 화면(액티비티)을 구성하는데 사용하는 요소

# 뷰
View 클래스를 상속해서 만든 위젯들

# 뷰그룹
ViewGroup 클래스를 상속해서 만든 위젯들
뷰들을 적절한 위치에 배치하거나 여러 뷰들을 하나로 모으는데 사용
배치 관리자:  뷰들을 적절한 위치에 배치
뷰 컨테이너: 여러 뷰들을 하나로 묶음

#텍스트뷰
사용자가 수정할 수 없는 텍스트를 표시하는 위젯
text
textSize: 단위는 sp(scale independent pixel: 해상도에 따라 크기 자동변환)
textColor: 16진수 코드로 표기
lines: 라인수 지정

# 에디트 텍스트
사용자에게 문자열이나 숫자를 입력받을 수 있는 위젯
inputType
layout_width
hint

# 버튼
사용자가 손으로 클릭할 수 있는 위젯
버튼을 통해 사용자가 어떤 결정을 선택하게 하거나
다음 화면으로 전환하는 등의 인터랙티브한 응답을 할 수 있음
backgroundTint: 버튼의 배경색
textAllCaps: 버튼 텍스트의 영어 대문자 표시 여부

# 이미지뷰
이미지를 화면에 보여주기 위한 위젯
src         : 이미지 경로 지정
scaleType   : 이미지를 어떻게 보여줄지 지정

# 체크박스
사용자가 여러 항목들 중 하나 이상의 항목을 선택할 수 있게 해주는 위젯
checked: 항목의 선택여부를 표시

# 라디오버튼
사용자가 여러 항목들 중 하나의 항목만을 선택할 수 있게 해주는 위젯
checked: 항목의 선택여부를 표시

# 토글버튼
사용자가 선택을 하면 켜짐이 표시되고 다시 선택하면 꺼짐을 표시하는 위젯
textOff : 토글버튼이 꺼졌을때 표시할 문자열 지정
textOn  : 토글버튼이 켜졌을때 표시할 문자열 지정

# 색상명 사용하기
위젯에 색상을 지정하려면 보통 16진수 코드를 이용함
단, 색상이름으로 색상을 지정하려면
values - colors.xml에서 색상명과 16진수 코드를 선언하고
''@color/색상명' 형태로 사용하면 됨

색상은 AABBCCDD 형태
(AA - 투명도, BB - 빨강, CC - 녹색, DD - 파랑) 또는

AABBCC 형태로 정의가능함
(AA - 빨강, BB - 녹색, CC - 파랑)
